# Templates

Put permanent reusable backgrounds and mockup images here.

Examples:

```text
templates/jewelry/pendant_neck_mockup_01.jpg
templates/jewelry/pendant_neck_mockup_01.json
templates/clocks/wall_clock_dark_room_01.jpg
templates/clocks/wall_clock_dark_room_01.json
```

Do not put normal product batches here. Product batches belong in `input/product_name/`.
